## 特性随笔

### 无条件循环

loop {} 等价于 while true {}，而未采用 do {} while 。感觉前者相对更容易使用。

### 字典包含某键

可用`字典.get(某键) != nil`判断。注意不能用`字典.get(某键)`, 因为值为 0 时也会返回假. 检查 python 的`键值 in 字典`的对应 ast，为 ast.In，未看到木兰对应操作。参考[ulgen](https://github.com/MulanRevive/mulan/blob/bc0ee27fc0dda93b5956999ec7f1aaf1061dd43a/ulang/codegen/ulgen.py#L456)，用了`__contains__`
