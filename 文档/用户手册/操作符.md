## 重载操作符语法范式

```
// 示例说明
type TypeName {
    func $TypeName(param) {
        self.value = param
    }
    
    // operator : OPERATOR binop ( param , param ) block
    operator 二元操作符 (参数) {
        // 实现部分
        return ...
    }
    
    // operator : OPERATOR uop op_none block
    operator 一元操作符 {
        // 实现部分
        return ...
    }
    
    // operator : OPERATOR - op_none block
    operator - {
        // 实现部分
        return ...
    }
}
```

```
二元操作符 : 操作符 加 操作数 块
          | 操作符 增量赋值 操作数 块
          | 操作符 減 操作数 块
          | 操作符 等于 操作数 块
```
